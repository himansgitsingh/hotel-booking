🏨 QuickStay – Hotel Booking / PG Website

QuickStay is a full-stack hotel and PG booking platform built using the MERN stack. It allows users to search, book, and manage accommodations with real-time availability and secure authentication.

🚀 Features
🔐 JWT-based Authentication (Login/Signup)
🏨 Browse hotels and PG listings
📅 Real-time availability tracking
📌 Booking management system
☁️ Cloud-based database (MongoDB Atlas)
⚡ RESTful APIs for backend operations
📊 Basic analytics (e.g., bookings, usage)
🛠️ Tech Stack

Frontend:

React.js
Tailwind CSS

Backend:

Node.js
Express.js

Database:

MongoDB (Atlas)

Other Tools:

JWT Authentication
Postman (API testing)
Git & GitHub
