🏨 QuickStay – Hotel Booking / PG Website

QuickStay is a full-stack hotel and PG booking platform built using the MERN stack. It allows users to search, book, and manage accommodations with real-time availability and secure authentication.
📂 Project Structure
QuickStay/
│── client/        # React frontend
│── server/        # Node.js backend
│── models/        # MongoDB schemas
│── routes/        # API routes
│── controllers/   # Business logic
│── middleware/    # Auth & error handling
│── config/        # DB config
